# SQLite-TXT-members
 
ToOMORROW X TOGETHERの公式サイトのメンバー情報を取得して、SQLiteを用いたプロフィールのデータベースの作成

# Requirement
 
Python 3.8.5
SQLite3

# Usage 
```bash
git clone git@github.com:nao1412/SQLite-TXT-members.git
cd SQLite-TXT-members
python txt.py
```
